Detailed Description

template<MIDIMessageType Type, uint8_t RangeLen>
class NoteCCKPRange< Type, RangeLen >

Generic base class for classes that listen for MIDI Note, Control Change and Key Pressure events on a range of addresses and save their values.

Template Parameters
TypeThe type of MIDI messages to listen for:
  • MIDIMessageType::NoteOn
  • MIDIMessageType::ControlChange
  • MIDIMessageType::KeyPressure
RangeLenThe length of the range of addresses to listen to.
