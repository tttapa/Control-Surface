◆ update()

Button::State Button::update ( )

Read the button and return its new state.

The button is debounced, the debounce time can be set in Settings.hpp: BUTTON_DEBOUNCE_TIME.

Raw input:

├────┤ ├────┤ ├─┼─┼────┤ ├─┼─┼────┤ ├─┼────┤
