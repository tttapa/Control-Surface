Detailed Description

A class of MIDIOutputElements that read the input of a momentary push button or switch, and send out MIDI Control Change events.

A value of 0x7F is sent when the button is pressed, and a value of 0x00 is sent when the button is released.
The button is debounced in software.
This version can be banked.

Create a new Bankable CCButton object with the given bank configuration, button pin, and address.

Parameters
configThe bank configuration to use: the bank to add this element to, and whether to change the address, channel or cable number.
pinThe digital input pin with the button connected.
The internal pull-up resistor will be enabled.
addressThe MIDI address containing the controller number [0, 119], channel [CHANNEL_1, CHANNEL_16], and optional cable number [0, 15].
senderThe MIDI sender to use.
