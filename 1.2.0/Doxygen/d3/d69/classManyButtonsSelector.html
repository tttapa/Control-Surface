Detailed Description

template<setting_t N>
class ManyButtonsSelector< N >

Selector that reads from \( N \) buttons.

Pressing the \( n \)-th button selects the \( n \)-th setting.

Template Parameters
NThe number of settings.
Examples
Program-Changer.ino.
