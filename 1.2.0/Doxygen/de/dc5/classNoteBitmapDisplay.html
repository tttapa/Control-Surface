Detailed Description

A class that displays a bitmap depending on the state of a MIDINote.

Examples
MCU-OLED-SSD1306-MEGA.ino, MCU-OLED-SSD1306-x2.ino, MCU-OLED-SSD1306.ino, and VU-Meter-Bridge.ino.
