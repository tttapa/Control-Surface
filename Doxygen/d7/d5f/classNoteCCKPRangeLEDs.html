Detailed Description

template<MIDIMessageType Type, uint8_t RangeLen>
class NoteCCKPRangeLEDs< Type, RangeLen >

Generic base class for classes that listen for MIDI Note, Control Change and Key Pressure events on a range of addresses and turn on the corresponding LED when the value is higher than a threshold.

Template Parameters
TypeThe type of MIDI messages to listen for:
  • NoteOn
  • ControlChange
  • KeyPressure
RangeLenThe length of the range of addresses to listen to.
