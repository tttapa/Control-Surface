Detailed Description

A class of MIDIOutputElements that read the input of a latching push button or toggle switch, and send out MIDI Control Change events.

When the switch changes state, two MIDI events are sent: the first one with a value of 0x7F, followed immediately by a second one with a value of 0x00.
The switch is debounced in software.
This version can be banked.

Create a new Bankable CCButtonLatching object with the given pin, the given controller number and channel.

Parameters
configThe bank configuration to use: the bank to add this element to, and whether to change the address, channel or cable number.
pinThe digital input pin with the button connected.
The internal pull-up resistor will be enabled.
addressThe MIDI address containing the controller number [0, 119], channel [Channel_1, Channel_16], and optional cable number [Cable_1, Cable_16].
senderThe MIDI sender to use.
