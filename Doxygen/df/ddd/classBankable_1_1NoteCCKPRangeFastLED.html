Detailed Description

template<MIDIMessageType Type, uint8_t BankSize, uint8_t RangeLen, class ColorMapper>
class Bankable::NoteCCKPRangeFastLED< Type, BankSize, RangeLen, ColorMapper >

Generic base class for classes that listen for MIDI Note, Control Change and Key Pressure events on a range of addresses and turns on the corresponding LED in a FastLED strip with a color that depends both on the active bank, the index in the range, and the value of the incoming MIDI message.

Template Parameters
TypeThe type of MIDI messages to listen for:
  • MIDIMessageType::NoteOn
  • MIDIMessageType::ControlChange
  • MIDIMessageType::KeyPressure
BankSizeThe number of banks.
RangeLenThe length of the range of addresses to listen to.
ColorMapperA callable that maps a 7-bit MIDI value, the bank index and the index in the range to a FastLED CRGB color, see Bankable::DefaultColorMapper for an example.

◆ onBankSettingChange()

template<MIDIMessageType Type, uint8_t BankSize, uint8_t RangeLen, class ColorMapper >
void onBankSettingChange ( )
inlineoverrideprotectedvirtual

A function to be executed each time the bank setting changes.

Think of an LED that indicates whether a track is muted or not. If this LED is bankable, let's say with 4 tracks per bank, 2 banks, and a base address of 3, then this LED object keeps the state of tracks 3 and 7. When the bank setting is 0, the LED displays the state of track 3, when the bank setting is 1, the LED displays the state of track 7. To know when to update the LED, this callback is used.
